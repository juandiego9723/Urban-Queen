\# Urban Queen



Sistema de ranking en tiempo real para dinámicas en TikTok Live.



\## Instalación



\### Requisitos

\- Node.js

\- Git



\### Clonar

```bash

git clone https://github.com/juandiego9723/Urban-Queen.git

cd Urban-Queen

```



\### Instalar dependencias

```bash

npm install

```



\### Ejecutar

```bash

node server.js

```



\### Abrir

http://localhost:3000
